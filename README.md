# DevOps Troubleshooting Journal

A collection of real-world troubleshooting cases, debugging steps, fixes, and lessons learned during my DevOps learning journey.

## Purpose

This repository is my personal troubleshooting knowledge base where I document:

- Linux issues
- Git problems
- Docker errors
- Jenkins troubleshooting
- Kubernetes debugging
- Networking issues
- CI/CD pipeline failures
- Vertual box & VM
- Server configuration problems

The goal is to improve problem-solving skills and maintain reusable solutions for future reference.

---

## Troubleshooting Format

Each troubleshooting document contains:

1. Problem Statement
2. Error Message
3. Root Cause Analysis
4. Investigation Steps
5. Solution/Fix
6. Verification
7. Lessons Learned
